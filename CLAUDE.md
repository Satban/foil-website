# CLAUDE.md - Foil Website

## Protected Files - DO NOT MODIFY

The following files have been manually designed and should NOT be overwritten or regenerated by any agent:

- `index.html` - Main homepage (forest green design, STUC animation, showcase sections)
- `values.html` - Values page (redesigned with forest green palette)

**If you need to update the homepage or values page, ask the user first.**

## Blog Agent Guidelines

When updating blog content:
1. Only modify files in `blog/` and `drafts/` directories
2. Add new posts to `blog/posts/`
3. Update `blog/index.html` to add new post links
4. Update `sitemap.xml` with new URLs
5. **Never regenerate or overwrite `index.html` or `values.html`**

## Design System

The site uses:
- **Colors**: Forest green palette (#1E3D32 primary, #2A5244 secondary, #E8836B coral accent)
- **Fonts**: Inter (sans), Fraunces (serif)
- **Icons**: Tabler Icons (webfont CDN)
- **Background**: Warm cream (#FEFBF6)

## Copy Guidelines

- Use positive framing ("use everything" not "waste nothing")
- Never use the word "waste" in copy
- Focus on "use" as the core action verb
- Tagline: "Know What. Know When."
